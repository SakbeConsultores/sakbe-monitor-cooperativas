IFC — Índice de Fragilidad Cooperativa
Modelo de alerta temprana Sakbé Consultores. Score 1,0 (máxima fragilidad) → 3,0 (máxima fortaleza). 5 dimensiones · 11 indicadores · umbrales regulatorios y sectoriales.
IFC = 0,30 × Solvencia + 0,25 × Calidad de activos + 0,20 × Liquidez + 0,15 × Rentabilidad + 0,10 × Eficiencia. Override: si solvencia regulatoria < 9%, IFC ≤ 2,29.
